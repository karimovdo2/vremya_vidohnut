# Audio placeholders (no binaries in repository)

В MVP используются ожидаемые пути аудио:
- `/sounds/rain.wav`
- `/sounds/sea.wav`
- `/sounds/forest.wav`

В текущем PR бинарные файлы звука удалены специально.
Для локальной проверки можно добавить собственные `.wav` файлы по этим именам в эту папку.
